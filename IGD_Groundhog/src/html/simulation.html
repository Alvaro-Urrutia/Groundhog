Daylight Factor

The Daylight Factor is an old static metric defined as the ratio of internal light level to external light level under a CIE Overcast Sky.

More info

It does not differentiate between climates or orientations.

The calculation program for this is RTrace. You can change the parameters in the Configuration panel.

Actual Illuminance

This option calculates the illuminance over the workplanes of the model using a CIE sky with the model's date, time and location.

More info

This is a Static metric that is meant to be used for date/time specific analysis.

Daylight Authonomy
