Welcome to Groundhog website

Groundhog is an Free and OpenSource plugin for creating Radiance Models within SketchUp. You are welcome to report bugs, add features, ask for new features, help us create documentation, provide a logo, etc.

An official version is still not released, but you are free to download the HEAD version, which has proven to be stable for us.

Installation

SketchUp allows installing extensions from the "preferences" --> "extensions"--> "Install extension" menu within SketchUp. The extensions come in an RBZ file, which is provided with all the versions of Groundhog. Instructions for doing this are all over the internet. You may have got Groundhog with the complete source code which includes an RBZ file, or only the RBZ file. Either case, you have to go to SketchUp and install the extension as usual.

Features

Groundhog developers think that the way a model is created in SketchUp makes sense in a deeper maner than only providing colors and shapes. That is, layers are separated for a reason, and components are created with a purpose. In addition to that, Groundhog also allows creating surfaces that are not meant to be confused with "normal surfaces". Specifically, Groundhog recognizes Windows as special surfaces (this is useful for doing daylighting analysis), as well as workplanes (which are exported as grids), illums and we are planning to introduce "solved workplanes" which are intended to show results. Accordingly, Groundhog allows:

  • Exporting complex polygons, not only triangles. This helps reducing the simulation time.
  • Easily working with thickness of walls.
  • Labeling surfaces as Windows, Workplanes and Illums.
  • Grouping windows.
  • Exporting Groups and Components as separate files that will be "xformed" in a main file.
  • Exportation is done by layer.
  • Exporting a "rif" file for easy rendering.
  • Storing Scenes (views) that will also be exported.
