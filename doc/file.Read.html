About this version

This is the first Developing (Head) version Upload. It worked for us until now, but it does not mean it is really stable. The previous stable version had to be completely reorganized in order to avoid trouble with other SketchUp plugins in use, so we hope this one works well.

Groundhog philosophy

The idea here is that people help on everything. From fixing the web-page and its tutorials, to enhancing the source code and proposing new ideas on what to add, and fill the 'To Do' section. This is why the whole project is in an online repository.
